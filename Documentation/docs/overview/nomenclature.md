Several standard typographical and variable name conventions are used within this Wiki and corresponding software and models:

## Typographical Conventions

|  |  |  |
| --- | --- | --- |
| *t* | Italic | Scalar |
| **a** | Lowercase Bold | Vector |
| **A** | Uppercase Bold | Matrix |

## Units
All models and software are presented in [SI Units](https://en.wikipedia.org/wiki/International_System_of_Units) (whether base, or derived), unless specified otherwise. 

**Note:** It is critical to ensure your units are consistent and understood by everyone when working in the realm of GNC. SI Units are [industry standard](https://www.space.com/3332-nasa-finally-metric.html), even in the USA, and really no other unit sets should be used in GNC (in my humble opinion).
